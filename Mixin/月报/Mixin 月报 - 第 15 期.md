### 其他
- 2020-05-17
  Mixin CEO 冯晓东直播：Mixin Network 的安全性、稳定性和优秀的 API 设计是币印选择其作为底层网络的关键，长达 1 年的合作对接过程中，币印团队反馈了很多改进建议，使得 API 变得更加完善；团队近期花了很多精力在提高稳定性，轻节点还没有进展，下一步会推动主网节点的奖励机制变更，而不是现在的收益均分，毕竟最初的设计就是按工作量来分配收益；Mixin 团队目前有 16 个人；Mixin Messenger 下一个大功能是端到端加密群语音功能，做 mornin 这个最初是想做一个例子来测试一下我们的技术方案是不是可行，短期内不会做视频聊天的功能，等以后用户非常多来再考虑；XIN 写在代码里唯一不变的作用就是做节点，总之牛逼之后用处就多。
